# 交接文件：小時光 官網（Little Moments）

> 給接手的工程師 / Claude Code：本資料夾是完整的設計交接包。目標是把 `design/` 內的 HTML 設計稿，**100% 還原**成一個可部署到 Vercel + Supabase + Railway 的正式網站，原始碼版控於 GitHub。

---

## 一、這是什麼

「小時光」是一個已有**線下書店門市**的品牌，線上要銷售：

1. **AI 藝術畫作**（生成後做成實體裱框畫作）— 可**租賃**、可**買斷**
2. **客製化旅行旅程** — 專屬顧問規劃
3. **會員制** — 加入會員以**點數**兌換旅程體驗與藝術品折扣
4. **線上預約參訪**（到店 / 到府顧問）

視覺方向：**「象牙沙龍 Ivory Salon」**——象牙白紙感、香檳金細線、優雅中文襯線，面向高單價藏家客群（董事長夫人等）。溫暖、克制、留白、精緻。

---

## 二、關於設計檔（重要）

`design/` 內的 `.dc.html` 是 **設計參考稿（用 HTML 製作的高擬真原型）**，用來精準呈現「最終外觀與互動行為」，**不是要直接複製上線的生產程式碼**。

- `小時光官網.dc.html` — **主交付**：完整六頁官網（頂部導覽切換頁面）。以此為還原基準。
- `小時光首頁探索.dc.html` — 早期三個首頁視覺方向的探索稿（僅供風格參考，非交付範圍）。

`.dc.html` 是一種會即時串流渲染的單檔格式，開啟即可在瀏覽器預覽。你**不需要**沿用這個格式；請用下方「建議技術棧」在正式框架中**重新實作**這些設計。

> 若要直接看設計：用瀏覽器開啟 `design/小時光官網.dc.html` 即可（需連網載入 Google Fonts）。

## 三、擬真度：High-fidelity（像素級）

本設計為 **hifi**。顏色、字體、間距、互動皆為最終樣式，請**像素級還原**。所有設計 token 見第七節。

## 四、頁面與畫面

主檔 `小時光官網.dc.html` 共 6 個頁面，透過頂部固定導覽（sticky header）切換。Header 左為 Logo「小時光」（點擊回首頁），中為導覽列，右為「預約參訪」外框按鈕。所有頁面共用同一個 header 與 footer。

### 1. 首頁 Home
- **目的**：品牌第一印象、導流至三大服務與會員。
- **版面（由上而下）**：
  1. **Hero**（置中）：斜體英文引言 `A private collection, curated for you`（Cormorant Garamond italic, 22px, 金 `#a2854a`）→ 44×1px 金色分隔線 → 主標「為懂得生活的人，典藏值得停留的時光」（Noto Serif TC 400, 64px, `#2a2016`, line-height 1.34, letter-spacing 0.05em, max-width 760px）→ 副標段落（16px, `#5a4d3b`, line-height 2.1, max-width 520px）→ 兩顆 CTA：實心深色「預約私人鑑賞」+ 底線文字「瀏覽典藏 →」。
  2. **主視覺圖組**：grid `1.4fr 1fr`，左一張大圖（高 440px），右直向兩張（各 min-height 216px），gap 2px。
  3. **三種開始的方式**（背景 `#f4ede0`）：置中小標 + 三欄卡片（gap 2px，分隔用 `#e2d7c4` 底色透出）。每卡：序號 01/02/03（Cormorant 34px 金）+ 標題（Noto Serif TC 500, 22px）+ 說明（14px）+ 底線連結。分別導向 藝術典藏 / 租賃買斷 / 私人旅程。
  4. **本季精選**：標題列（左標題 34px、右「看全部作品 →」）+ 三欄作品卡（圖 300px + 品名 + 租/買價）。
  5. **會員 CTA 區**（深色 `#2e2519`, 文字 `#f4ede0`）：左標題與說明，右「申請入會」外框按鈕（金框）。
- **CTA 導向**：預約私人鑑賞→預約頁；瀏覽典藏→藝術典藏；三卡→對應頁；申請入會→會員沙龍。

### 2. 藝術典藏 Gallery
- **目的**：瀏覽所有可租/可買的 AI 藝術畫作。
- **版面**：頁首（斜體 `The Collection` + 主標「藝術典藏」52px + 說明）→ 篩選標籤列（全部 / 風景 / 抽象 / 靜物 / 植物；「全部」為深色實心 active，其餘為 `#d8c9a8` 外框）→ **三欄作品網格**（gap 32px×28px）。
- **作品卡**：圖 320px → 品名（Noto Serif TC 19px）+ 右上媒材（Cormorant 14px 金，如 `Giclée`/`Canvas`）→ 價格（12.5px `#8a7259`，「租賃 NT$X/月 · 買斷 NT$Y」）。
- **資料**：9 筆（見 `小時光官網.dc.html` logic 的 `works` 陣列，含 title / tag / medium / price）。篩選標籤目前為靜態視覺；正式版應以 `tag` 過濾。

### 3. 私人旅程 Journeys
- **目的**：呈現以畫作靈感延伸的客製旅程，並可用點數兌換。
- **版面**：全幅主視覺（高 460px，深色漸層覆蓋，底部疊斜體 `Bespoke Journeys` + 主標「把一幅畫，延伸成一趟旅行」54px 白）→ 導言段落 → **三列旅程**（每列 grid `0.9fr 1.5fr 0.6fr`：縮圖 / 標題+說明 / 右側價格）。
- **三筆旅程**：京都（NT$128,000 或 6,400 點）、托斯卡尼（NT$268,000 或 13,400 點）、北海道（NT$156,000 或 7,800 點）。價格區塊：`from`（Cormorant 金）+ 金額（Noto Serif TC 24px）+「或 N 點」。
- 底部 CTA「預約專屬旅程諮詢」→ 預約頁。

### 4. 租賃 · 買斷 Rental
- **目的**：說明營運模式（租賃 / 先租後買 / 買斷）。
- **版面**：置中頁首 → **三方案卡**（max-width 1180）：
  - **月租方案** — NT$580 起/月；每季換畫、含裱框運送安裝、保養保險、可升級買斷。
  - **先租後買**（深色卡 `#2e2519`，左上「最受歡迎」金標籤）— NT$780 起/月；月租金 100% 折抵買斷、12 個月內買斷免手續費、會員點數回饋。
  - **直接買斷** — NT$9,600 起；永久擁有附收藏證書、會員折扣最高 15%。
  - 每卡：小標（Cormorant 大寫）+ 中文標題（Noto Serif TC 500, 26px）+ 價格（34px）+ 分隔線 + 條列（· 開頭，14px）。
- 底部「到府顧問」橫幅（`#f4ede0`）+ CTA「預約到府顧問」→ 預約頁。

### 5. 會員沙龍 Membership
- **目的**：會員分級、點數規則、兌換範例。
- **版面**：深色頁首（`#2e2519`）→ **三級會員卡**（max-width 1180）：
  - **緻銀 Silver** — 每 NT$100 累 1 點；買斷 95 折；生日雙倍點數；電子選畫刊物。
  - **璀金 Gold**（`#f4ede0` 卡，頂部置中「推薦」金標籤）— 每 NT$100 累 1.5 點；買斷 9 折 / 租賃 95 折；每季私人鑑賞會；旅程兌換優先。
  - **典藏 Platinum** — 每 NT$100 累 2 點；買斷 85 折 / 租賃 9 折；專屬顧問一對一；私人旅程年度席位。
  - 每卡置中：等級英文（Cormorant）+ 中文名（Noto Serif TC 26px）+ 條列（左對齊，13px）。
- **點數兌換一覽**：四欄（gap 2px）：500pt→租賃折 NT$500；1,200pt→私人鑑賞午茶席；6,400pt→京都旅程折抵；13,400pt→托斯卡尼旅程折抵。點數數字為 Cormorant 30px 金。

### 6. 預約參訪 Booking
- **目的**：預約到店 / 到府，並收單。
- **版面**：grid `1fr 1fr`。
  - **左**：斜體 `Book a private visit` + 主標「預約參訪」48px + 導言 + 門市空間圖（280px）+ 三段資訊（Address / Hours / Enquiry，小標為 Cormorant 大寫金）。
  - **右表單卡**（`#f4ede0`, padding 52×48）：稱謂姓名、聯絡電話+預約人數（兩欄）、**參訪目的**（四個可點選 chip：鑑賞畫作 / 租賃·買斷 / 規劃旅程 / 入會諮詢，選中變深色實心）、期望日期（date）+ 期望時段（select）、備註（textarea）、送出按鈕。
  - 表單欄位：label 12.5px `#8a7259`，input padding 13×16、外框 `#d8c9a8`、底 `#faf6ee`。
  - **互動**：點「送出預約」後按鈕文字變「已收到您的預約，我們將盡快聯繫 ✓」（見 state）。

## 五、互動與行為

- **頁面切換**：SPA 式，點導覽 / CTA 切換頁面內容，切換時 `window.scrollTo({top:0})`。正式版建議改為真實路由（見技術棧）。
- **導覽 active 態**：目前頁在導覽項下加 `2px solid #bfa06a` 底線。
- **連結 hover**：`a:hover` 目前無特別樣式（探索稿為 opacity 0.6）；正式版建議 hover 轉金色 `#9a7d47` 或加底線。
- **參訪目的 chip**：單選，選中 → 背景 `#2e2519`、字 `#f4ede0`、框 `#2e2519`；未選 → 框 `#d8c9a8`、字 `#6a5a44`、底 `#faf6ee`。
- **預約送出**：目前為前端 state 切換文案（原型無後端）。正式版須寫入資料庫並寄信/通知。
- **篩選**：藝術典藏標籤正式版應真正過濾。

### 響應式（RWD）
桌機優先，三個斷點：
- **≤1024px**：容器左右內距縮為 40px；h1 縮為 44px。
- **≤820px**：所有多欄 grid 收成單欄（gap 24px）；header 換行、導覽列改為可橫向滑動（overflow-x auto）；容器內距 22px；h1 34px、h2 26px；大區塊上下內距縮小。
- **≤520px**：導覽 gap 縮小；主標降至 27–29px。

## 六、狀態管理（原型）

主檔 logic（`class Component`）持有：
- `page`：目前頁（`home` / `gallery` / `journeys` / `rental` / `membership` / `booking`）。
- `purpose`：預約表單選中的參訪目的。
- `booked`：是否已送出預約（控制按鈕文案）。
- 衍生：`isHome…isBooking`（頁面顯示）、`navHome…navMembership`（導覽底線）、`works`（作品）、`purposes`（chip）、`bookingBtn` / `submitBooking`。

> 正式版狀態請改用框架路由 + 伺服器資料（Supabase）。原型的 `works` / 旅程 / 方案 / 會員級距數值可直接作為 seed 資料。

## 七、設計 Tokens

### 字體
| 用途 | 字體 | 備註 |
|---|---|---|
| 中文標題 | **Noto Serif TC**（300–700） | 主標、卡片標題 |
| 西文點綴 | **Cormorant Garamond**（400/500，含 italic） | 引言、序號、價格 from、大寫小標 |
| 內文 / 導覽 / 表單 | **Noto Sans TC**（300/400/500） | body 預設 |

載入：`https://fonts.googleapis.com/css2?family=Noto+Serif+TC:wght@300;400;500;600;700&family=Cormorant+Garamond:ital,wght@0,400;0,500;0,600;1,400;1,500&family=Noto+Sans+TC:wght@300;400;500&display=swap`

常用字級：主標 64/54/52/48px；區塊標題 34–36px；卡片標題 22–26px；內文 14–16px；小標籤 12–13px。標題常用 `letter-spacing:0.04–0.05em`，Logo `0.2em`。

### 顏色
| 用途 | Hex |
|---|---|
| 頁面底（象牙白） | `#faf6ee` |
| 次要區塊底 | `#f4ede0` |
| 深色區塊 / 主墨字 | `#2e2519` |
| 主要文字（更深） | `#2a2016` |
| 導覽 / 一般文字 | `#4d4132` |
| 內文次要 | `#5a4d3b` |
| 說明淺字 | `#7a6b52` / `#8a7259` |
| 深底上的內文 | `#c9bca6` / `#d6c9b0` |
| 金（主）accent | `#bfa06a` |
| 金（深，文字） | `#9a7d47` / `#a2854a` |
| 金（亮，深底上） | `#e3c98f` |
| 金色細線 hairline | 漸層 `transparent → #d8c9a8 → transparent` |
| 邊框 / 分隔 | `#e2d7c4` / `#d8c9a8` |
| 佔位圖底 | `#ece1cd`（135° 斜紋 `rgba(120,95,70,0.09)`） |

### 間距 / 形狀
- 容器 max-width：`1360px`（主要）、`1180px`（方案 / 會員卡）。
- 區段內距（桌機）：上下 80–104px、左右 64px。
- 卡片內距：34–48px。多欄間距：多用 `gap:2px` 配底色透出，或 28–32px。
- 圓角：整體極少圓角（≈2px 或 0），走硬邊精緻感。
- 陰影：頁面本身幾乎不用陰影（探索稿卡片用 `0 40px 90px rgba(60,45,30,0.24)`，正式版首頁不需要）。
- 斷點：1024 / 820 / 520px。
